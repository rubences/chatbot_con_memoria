# Guía rápida de Django

Django separa responsabilidades entre modelos, vistas y plantillas.

Temas frecuentes:
- ORM para consultas y persistencia.
- Migraciones para versionar esquema de base de datos.
- URLs para enrutar peticiones.
- Seguridad: CSRF, validación de formularios y gestión de sesiones.

Recomendación:
- Mantener la lógica de negocio fuera de las vistas y usar servicios/orquestadores.
