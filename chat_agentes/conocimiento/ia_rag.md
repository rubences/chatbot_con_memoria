# Guía rápida de IA y RAG

RAG combina recuperación de contexto con generación de texto.

Temas frecuentes:
- Enrutamiento por intención o temática.
- Recuperación de fragmentos relevantes desde archivos.
- Prompts de sistema para especializar agentes.
- Trazabilidad: citar fuentes usadas para responder.

Recomendación:
- Si hay ambigüedad temática, activar varios agentes y fusionar respuestas.
