# Guía rápida de Python

Python prioriza legibilidad y simplicidad.

Temas frecuentes:
- Estructuras: listas, diccionarios, conjuntos y tuplas.
- Funciones con type hints y docstrings claros.
- Pruebas con pytest y buenas prácticas PEP 8.
- Manejo de errores con excepciones específicas.

Recomendación:
- Usar nombres descriptivos, funciones pequeñas y validar entradas.
