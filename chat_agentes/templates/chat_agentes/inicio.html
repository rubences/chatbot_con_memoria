Colaboracion multiagente

El sistema combina especializacion por roles y orquestacion central. Estas son las estrategias clave:

Basada en reglas

Flujo predecible con decisiones por reglas explicitas. Ideal para procesos estables y auditables.

Basada en roles

Cada agente tiene una responsabilidad concreta y colabora con contexto compartido para un objetivo comun.

Basada en modelos

Razonamiento adaptativo ante incertidumbre. Aporta flexibilidad, con mayor complejidad computacional.

Metodologia ReWOO

Separa planificacion, uso de herramientas y sintesis para mejorar precision y eficiencia de tokens.
